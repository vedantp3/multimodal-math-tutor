# JEE Tips & Tricks — Algebra

## Quick Calculation Tricks
- **Vieta's Formulas**: For ax² + bx + c = 0, sum of roots = -b/a, product of roots = c/a. Use to find root expressions without solving.
- **Symmetric expressions**: If a problem asks for α² + β², use (α + β)² - 2αβ.
- **Telescoping sums**: Look for cancellation patterns in series like Σ 1/(n(n+1)) = Σ (1/n - 1/(n+1)).

## Pattern Recognition
- **Recurring patterns in JEE**: Quadratic in disguise (e.g., x⁴ + x² + 1 is quadratic in x²).
- **Substitution**: For expressions like x + 1/x, try t = x + 1/x, then x² + 1/x² = t² - 2.
- **AM-GM shortcut**: When maximizing a product with fixed sum, or minimizing a sum with fixed product, use AM ≥ GM directly.

## Common JEE Algebra Question Types
1. **Find the number of real roots** — Use discriminant and sign analysis
2. **Solve exponential/logarithmic equations** — Take log or exponentiate both sides
3. **Series sum problems** — Identify as AP, GP, AGP, or telescoping
4. **Functional equations** — Try f(0), f(1) to find pattern, then verify
5. **Inequality proofs** — AM-GM, Cauchy-Schwarz, or triangle inequality

## Time-Saving Strategies
- In MCQs, substitute answer choices to verify
- Use options to eliminate: if discriminant < 0, eliminate real-root answers
- For "find the value" questions, try plugging in specific numbers first
