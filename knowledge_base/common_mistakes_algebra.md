# Common Mistakes in Algebra

## Mistake 1: Sign Errors in Expansion
**Wrong**: (a - b)² = a² - b²
**Correct**: (a - b)² = a² - 2ab + b²
**Tip**: Always expand step by step, never skip the middle term.

## Mistake 2: Incorrect Factoring
**Wrong**: a² + b² = (a + b)(a - b)
**Correct**: a² - b² = (a + b)(a - b). Note: a² + b² cannot be factored over reals.
**Tip**: Check if the expression is a difference of squares, not a sum.

## Mistake 3: Dividing by Variable Without Considering Zero
**Wrong**: If x² = 3x, then x = 3.
**Correct**: x² - 3x = 0 → x(x - 3) = 0 → x = 0 or x = 3.
**Tip**: Never divide both sides by a variable unless you know it's non-zero.

## Mistake 4: Wrong Application of Logarithm Rules
**Wrong**: log(a + b) = log(a) + log(b)
**Correct**: log(ab) = log(a) + log(b). log(a + b) has no simple form.
**Tip**: Logarithm rules only apply to multiplication, division, and powers — never to addition.

## Mistake 5: Forgetting Domain Restrictions
**Wrong**: Solving √x = -3 gives x = 9.
**Correct**: No solution. √x is always ≥ 0.
**Tip**: Always check if your solution is valid in the original equation's domain.

## Mistake 6: Incorrect Inequality Direction
**Wrong**: If -2x > 6, then x > -3.
**Correct**: If -2x > 6, then x < -3 (inequality flips when dividing by negative).
**Tip**: Always flip the inequality sign when multiplying or dividing by a negative number.

## Mistake 7: GP Sum Formula Misapplication
**Wrong**: Using S = a/(1-r) for |r| ≥ 1.
**Correct**: The infinite GP sum formula only works when |r| < 1.
**Tip**: Always check if the series converges before applying the infinite sum formula.
