# Calculus Solution Templates

## Template: Evaluating Limits
**Step 1**: Try direct substitution first
**Step 2**: If 0/0 or ∞/∞ form, try algebraic simplification (factoring, rationalizing, etc.)
**Step 3**: If still indeterminate, apply L'Hôpital's Rule: differentiate numerator and denominator separately
**Step 4**: Re-evaluate the limit
**Step 5**: Verify with known standard limits
**Check**: Ensure the limit exists (check left and right limits for two-sided limits)

## Template: Finding Derivatives
**Step 1**: Identify the type of function (polynomial, trig, exponential, composite, etc.)
**Step 2**: Apply the appropriate rule (power rule, chain rule, product rule, quotient rule)
**Step 3**: Simplify the result
**Step 4**: Verify by checking dimensions/units or with a specific point
**Note**: For composite functions, always identify the outer and inner functions before applying chain rule

## Template: Maxima/Minima Problems (Optimization)
**Step 1**: Define the function to optimize and identify constraints
**Step 2**: Express the function in terms of one variable using constraints
**Step 3**: Find f'(x) and set f'(x) = 0 to find critical points
**Step 4**: Apply second derivative test: f''(x) > 0 → minimum, f''(x) < 0 → maximum
**Step 5**: Check boundary values if the domain is closed
**Step 6**: State the maximum/minimum value
**Check**: Verify the answer is physically reasonable given the problem context

## Template: Definite Integration
**Step 1**: Identify the integrand and limits of integration
**Step 2**: Find the antiderivative F(x) using appropriate technique
**Step 3**: Apply the Fundamental Theorem: F(b) - F(a)
**Step 4**: Simplify
**Techniques**: Substitution, integration by parts, partial fractions, trig substitution

## Template: Related Rates
**Step 1**: Draw a diagram and label variables
**Step 2**: Write an equation relating the variables
**Step 3**: Differentiate both sides with respect to time t
**Step 4**: Substitute known values and solve for the unknown rate
**Step 5**: Include units in the answer
